### Configuration and Installation

1) Create the venv

```
python3 -m venv .venv
```

2) Activate venv

```
source .venv/bin/activate
```

3) Installation of requirements

```
pip install -r requirements.txt
```

4) Copy of example settings

```
cp .env.example .env
```

### Run the RetroGear - Engine


```
python3 -m src.main
```
